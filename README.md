# LogLens

A Python-based Linux log analyzer that parses syslog files, detects errors and warnings, and surfaces patterns in system logs.

## Features (v1)

- Parses standard Linux syslog format
- Identifies log severity levels (ERROR, WARNING, INFO)
- Groups logs by source process
- Provides clean structured CLI output using the rich library

## Usage
python cli.py --file samples/sample1_syslog.log
